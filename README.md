# 📊 Superstore Sales Dashboard using Microsoft Excel

## 📌 Project Overview

The **Superstore Sales Dashboard** is an interactive Business Intelligence (BI) dashboard built using **Microsoft Excel**. It analyzes retail sales data and provides meaningful insights into sales performance, profitability, customer segments, regional sales, and product performance using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

The dashboard helps users monitor key business metrics and interactively explore the data through filters, making it easier to support data-driven decision-making.

---


## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor key business KPIs.
- Compare sales across different regions.
- Identify top-performing products.
- Analyze customer segments.
- Track monthly and yearly sales trends.
- Build an interactive dashboard for business analysis.
- Support data-driven decision-making.

## 💡 Key Insights

- Total sales exceeded **₹2.29 million**.
- The business generated a **total profit of ₹286,397**.
- **West** and **East** regions recorded the highest sales.
- **Technology**, **Furniture**, and **Office Supplies** were the major revenue-generating categories.
- Monthly sales increased toward the end of the year.
- Top-selling products contributed significantly to total revenue.
